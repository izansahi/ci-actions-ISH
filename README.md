# Ejercicio 1 - Documentación con JSDoc

Este proyecto demuestra la documentación de código JavaScript usando JSDoc.

## Características

- **Calculator**: Clase para operaciones matemáticas básicas
- **StringUtils**: Clase utilitaria para operaciones con cadenas

## Instalación

```bash
npm install
```

## Uso

```bash
npm start
```

## Generación de documentación

```bash
npm run docs
```

La documentación se generará en la carpeta `docs/`.
